# 📊 Trading Landing Page

This is a simple landing page created to redirect users from ads to a Telegram trading community.

## 🚀 Purpose
- Bridge page for Facebook Ads
- Redirect users to Telegram channel
- Provide basic info about trading community

## 📌 Features
- Clean and simple UI
- Mobile responsive (Bootstrap)
- Fast loading
- Direct CTA button for Telegram
